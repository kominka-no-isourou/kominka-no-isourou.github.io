# 古民家の記録 — isourou

築およそ100年、かつて宿場であり、炭鉱の街で石炭を運ぶ拠点でもあった古民家。
大家さんと少しずつ受け継ぎながら、その今の姿・出来事・進み具合を残していくサイトです。

このリポジトリは **isourou** のみんなで管理しています。
プログラミングに詳しくなくても大丈夫。下の手順どおりに触れば、写真や文章を足していけます。

---

## 📁 ファイルの構成

```
index.html        … サイト本体（1ファイル）
images/           … 写真を入れるフォルダ
写真の入れ方.txt   … 写真・文章の直し方（詳しい版）
```

---

## 🖼 写真を足す・差し替える

1. `images` フォルダを開く
2. 下の名前で写真を入れる（`.jpg` でも `.png` でもOK。名前を合わせるだけ）

| 場所 | ファイル名 |
|------|-----------|
| トップのカルーセル（6枚・自動でゆっくり切替） | `hero-1.jpg` 〜 `hero-6.jpg` |
| 古民家の今 | `now-1.jpg` 〜 `now-4.jpg` |
| 記録・進捗 | `record-1.jpg` |
| イベント | `event-1.jpg`, `event-2.jpg` |
| 訪れる（下部の背景） | `visit.jpg` |

- 写真が無い場所は、自動で「写真をここに」の枠が出ます。あとから足せます。
- 大きすぎる写真は重くなるので、**横幅2000pxくらいに縮小**してから入れると軽快です。

### GitHub上だけで写真を足す方法（パソコンに詳しくない人向け）
1. `images` フォルダを開く → 右上の **Add file → Upload files**
2. 写真をドラッグして **Commit changes** を押すだけ

---

## ✏️ 文章を直す

`index.html` の中の

```
<!-- ▼ここから -->  〜  <!-- ▲ここまで -->
```

の間の日本語を書き換えるだけです。

### GitHub上だけで直す方法
1. `index.html` を開く → 右上の **鉛筆アイコン（Edit）**
2. 直したら **Commit changes**

---

## 🌐 公開のしかた（GitHub Pages・無料）

1. リポジトリの **Settings → Pages** を開く
2. **Source** を `Deploy from a branch` にする
3. Branch を `main` / フォルダを `/ (root)` にして **Save**
4. 数分待つと公開されます

保存（Commit）するたびに自動で更新されます。

---

## 🔧 ちょっとした調整メモ

- カルーセルの切替の速さ … `index.html` の `9000`（ミリ秒＝表示時間）を変える
- 切替のゆっくりさ … `opacity 2.4s`（フェード時間）を変える
- 問い合わせ先・SNSのリンク … 「訪れる」セクションの `<a href="#">` の `#` を、
  実際のメール（`mailto:xxx@xxx`）やSNSのURLに変える
